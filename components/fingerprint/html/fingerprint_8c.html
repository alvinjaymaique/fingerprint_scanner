◆ fingerprint_set_command()

esp_err_t fingerprint_set_command ( FingerprintPacket * cmd,
uint8_t command,
uint8_t * params,
uint8_t param_length )

Sets or modifies a fingerprint command packet.

This function initializes or updates a FingerprintPacket with the specified command and parameters. It ensures that the parameter length does not exceed 4 bytes and automatically calculates the checksum. This function is useful for preparing commands before sending them to the fingerprint module.

Parameters
[out]cmdPointer to the FingerprintPacket to be populated or modified.
[in]commandCommand byte to be set in the packet.
[in]paramsPointer to a parameter array (can be NULL if no parameters).
[in]param_lengthNumber of parameters (max 4).
Returns
  • ESP_OK on success.
  • ESP_ERR_INVALID_ARG if cmd is NULL.
  • ESP_ERR_INVALID_SIZE if param_length exceeds 4.

< Adjust based on the largest required parameter size.

◆ g_fingerprint_event_handler

fingerprint_event_handler_t g_fingerprint_event_handler = NULL

Global pointer to the fingerprint event handler function.

This pointer is used to assign a function that will handle fingerprint-related events. The function should be defined by the user and should match the fingerprint_event_handler_t callback type. The event handler is called whenever a fingerprint-related event occurs, such as detection, image capture, or match results.

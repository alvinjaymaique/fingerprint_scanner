void fingerprint_build_command ( FingerprintCommand * cmd,
uint8_t command,
uint8_t * params,
uint8_t param_length )

Builds a fingerprint scanner command.

This function constructs a command packet by filling in the required fields such as the header, address, command, and parameters. The checksum is calculated automatically.

Parameters
[out]cmdPointer to the FingerprintCommand structure to be populated.
[in]commandCommand ID to be set in the packet.
[in]paramsPointer to an array of command parameters (if any).
[in]param_lengthLength of the parameters array (should be <= 4).

◆ fingerprint_set_baudrate()

void fingerprint_set_baudrate ( int baud)

Sets the baud rate for fingerprint module communication.

This function adjusts the baud rate used for UART communication with the fingerprint scanner. It should be called before fingerprint_init() to ensure proper communication settings.

Parameters
[in]baudThe desired baud rate (e.g., 9600, 57600, 115200).

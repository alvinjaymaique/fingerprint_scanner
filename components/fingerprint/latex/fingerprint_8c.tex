\item If a fingerprint is detected, generate the first fingerprint template ({\ttfamily PS\+\_\+\+Gen\+Char1}).
\item Prompt the user to remove their finger and wait for confirmation of removal.

\item If a finger is not detected within the allowed retries, the function will return an error.
\item If the finger is not removed within a timeout period, the process will restart.
\item If enrollment fails at any step, it retries up to three times before failing.
\end{DoxyItemize}

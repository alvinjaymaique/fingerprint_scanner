Initiates the fingerprint enrollment process.\+This function starts the automatic fingerprint enrollment process by sending the {\ttfamily PS\+\_\+\+Auto\+Enroll} command to the fingerprint module. The user is required to scan their finger multiple times for a successful enrollment.


\begin{DoxyParams}{Parameters}
{\em fingerprint\+\_\+id} & The unique ID to assign to the enrolled fingerprint. \\
\hline
{\em num\+\_\+scans} & The number of times the user must scan their finger for enrollment.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
The enrollment process requires the user to place and remove their finger multiple times until the sensor successfully registers the fingerprint template.
\end{DoxyNote}
\begin{DoxyReturn}{Returns}
None. The function triggers an event upon enrollment success or failure.
\end{DoxyReturn}
